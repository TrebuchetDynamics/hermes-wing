# Contributing

Hermes Wing is a Hermes-only Flutter alpha. Keep product language and routes within
the Hermes session/run model described in `CONTEXT.md` and `docs/adr/`.

## Setup

Use Flutter 3.44.2 and Node.js 22.

```bash
flutter pub get
npm ci
```

## Required checks

```bash
dart format --output=none --set-exit-if-changed lib test integration_test
flutter analyze
flutter test --concurrency=1
flutter build web --release -t lib/main_e2e.dart
npm run web:e2e
npm audit
```

Add the smallest regression test that proves non-trivial behavior. Never place
API keys, transcripts, private endpoint URLs, or generated agent state in a
commit. `.pi`, build output, screenshots generated by tests, and local tool
state remain ignored.

## Pull requests

Describe the user-visible change, security or privacy impact, platforms tested,
and exact validation commands. Compilation alone does not establish platform
support. Do not claim offline speech, release readiness, or signed distribution
without corresponding runtime evidence.
