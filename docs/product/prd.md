# Navivox Hermes PRD

Navivox connects Flutter web/desktop/mobile clients to a trusted Hermes Agent API server. The primary user journey is: enter endpoint URL, optionally enter API key, connect, choose or create a Hermes session, send text or local voice transcripts, review tool progress, answer approvals, and stop active runs when needed.

Non-goals: legacy gateway setup, profile-contact management, external memory dashboards, and server configuration editors.
